# nooise

Fluid focus music for the terminal.

## Requires Rust

Install Rust with rustup:

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Open a new terminal after rustup finishes.

## Install nooise

```sh
cargo install nooise --locked
```

## Start

```sh
nooise
```

Press `q` to quit.

## From Source

```sh
cargo run
```
